Core4 Concept Technical Definition Real-Life Analogy Python Example Scratch Example
Variables Reserved memory locations to store values In Scratch, you can make variables for different players to distinguish their different functions throughout a game.
Loops Script that allows you to repeat sections of code In Scratch, you can use loops to repeat a Sprite's movement so that you won't have to copy and paste the same actions over and over.
Conditionals A statement that controls the flow of execution based on some condition. In Scratch, you can use conditionals for games. You can make the game end if a sprite were to touch another.
Functions A group of related statements that perform a certain task In Scratch, you can use a function to define and simplify the modes of certain actions to a single block.
